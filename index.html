Sobre

O MEFWEB é um programa educacional gratuito de cálculo estrutural através do Método de Elementos Finitos, dirigido tanto para estudantes de Engenharia Civil como Engenheiros de Estruturas e pós-graduação.

Software MEFWEB

Tem por objetivo projetar e construir um programa computacional de uso livre (freeware) para ser usados como elementos pedagógicos na aprendizagem de análise e dimensionamento de estruturas, assim como elemento efetivo de uso profissional para atualização permanente da construção do conhecimento na área de estruturas e áreas correlatas.

Ferramenta
Educacional

O software é utilizado na disciplina de metodos dos elementos finitos, onde o aluno pode realizar análises de pórticos e bidimensionais, possibilitando universidades e institutos gerando conhecer e utilizar a ferramenta.

2

Software
Gratuito

É uma poderosa ferramenta educacional que auxilia nas análises de pórticos e bidimensionais, plataforma online possibilitando a facilidade acesso para professores e docentes.

3

Interface
Intuitiva

Com uma interface gráfica de usuário muito simples, intuitiva e fácil uso, e foi concebido para atender atividades de educacional, bem como projetos de engenharia de estruturas e com geração de relatorios de atividades desenvolvida.
